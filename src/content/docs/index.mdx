    #### Pattern everything
    TidalCycles allows you to make patterns with code. It includes language for describing flexible (e.g. polyphonic, polyrhythmic, generative) sequences of sounds, notes, parameters, and all kind of information.

    ```haskell
    d1
      $ jux (rev)
      $ sound "bd sn:1"
    ```
